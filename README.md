# 💬 Python Chat Application

A real-time chat application built using Python and socket programming.  
This project demonstrates a simple client-server architecture where multiple users can communicate through the command line in real time.

---

## 🚀 Features

- Real-time messaging
- Client-server architecture
- Multiple users can join the chat
- Command-line based interface
- Lightweight and easy to run
- Uses socket programming
- Multi-threading support for handling multiple clients

---

## 🛠️ Technologies Used

- Python
- Socket Programming
- Threading
- Command Line Interface (CLI)
